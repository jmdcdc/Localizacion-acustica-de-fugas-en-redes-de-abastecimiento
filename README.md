En España operan varios fabricantes de aparatos localizadores de fugas mediante técnicas acústicas, que permiten monitorizar las pérdidas de agua en las redes de abastecimiento. Los localizadores envian de forma periódica a las plataformas de su fabricante los datos registrados (cantidad de ruido detectado, probabilidad de fuga, localización aproximada de la fuga en sus inmediaciones e incluso el archivo de sonido con la fuga). Cada fabricante opera con su propia plataforma y es incompatible con el resto.
Los cuadernos de Jupyter permiten la descarga de los datos desde las API que implementan las plataformas de dos soluciones comerciales que se pueden encontrar actualmente en España. Los aparatos se comercializan en todo el mundo, y aunque las direcciones de las API podrían variar para otros mercados, el funcionamiento del código sería compatible.
Los datos extraidos incluyen archivos con el posicionamiento de los localizadores y/o las fugas detectadas, por lo que los historicos de cada incidente pueden llevarse sobre una solución como QGIS para facilitar la conservación de un único histórico y el análisis de incidentes sobre polígonos de corte (conjunto de tuberías que comparten válvulas de cierre que permiten aislarlas del resto de la red para proceder a su reparación), estudiar el envejecimiento de las infraestructuras y planificar inversiones de renovación, tanto de las calles donde se han producido incidentes, como de todas las que quedaron afectadas por las variaciones de presión durante las actuaciones.
También se adjuntan 2 scripts para QGIS que permiten cargar los dataloggers de cada fabricante directamente en una capa o shape y visualizarlos en el mapa. Los campos de las tablas varían segun los que facilita el fabricante para la consulta efectuada, que se corresponde con la primera lanzada en los cuadernos de Jupyter.
